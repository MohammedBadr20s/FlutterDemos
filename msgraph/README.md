# MSGraph Flutter

MSGraph Flutter is a Demo of Microsoft Graph Native APIs Integration with flutter using Method Channel to get the data from native Libs in Android And IOS

## Installation

```bash
git clone the project
flutter pub get
run
```

## Usage

```python
1- Run the Project
```
